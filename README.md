# Online-Shopping
Spring Boot kullanılarak yazılan Java tabanlı e-ticaret web uygulamasıdır .Geliştirdiğim e-ticaret platformunda Spring Boot (Backend) ve React (Frontend) teknolojilerini kullandım. PostgreSQL veritabanında ilişkisel yapı kurarak ürün, sipariş ve kullanıcı yönetimi sağladım.

Özellikler:

-Ürün, kategori, müşteri, sipariş, sepet tabloları

-Sepete ürün ekleme, silme, sipariş tamamlama

-React frontend ile modern arayüz
