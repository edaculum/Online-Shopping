# Online-Shopping
Spring Boot kullanılarak yazılan Java tabanlı e-ticaret web uygulamasıdır .Geliştirdiğim e-ticaret platformunda Spring Boot (Backend) ve React (Frontend) teknolojilerini kullandım. PostgreSQL veritabanında ilişkisel yapı kurarak ürün, sipariş ve kullanıcı yönetimi sağladım. Katmanlı mimari (Controller, Service,Repository) yapısını kullanarak projede daha anlaşılır, bakımı kolay ve genişletilebilir bir yapı kurmayı hedefledim. Ayrıca Spring Data JPA ile veri tabanı işlemlerini kolaylaştırdım

Özellikler:

-Ürün, kategori, müşteri, sipariş, sepet tabloları

-Sepete ürün ekleme, silme, sipariş tamamlama

-React frontend ile modern arayüz
